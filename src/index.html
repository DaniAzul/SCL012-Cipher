Golocódigo

El código Cesar o cifrado por desplazamiento es una
técnica sencilla de cifrado que podemos usar para
generar pistas para juegos con nuestras niñas.
Consiste en sustituir cada letra del texto original por
la letra que se encuentra a determinado número de
posiciones de la original.
